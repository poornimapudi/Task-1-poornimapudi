# Simple To-Do List Application

A beginner-friendly command-line To-Do List application built with Python. This project allows users to manage their daily tasks by adding, viewing, and removing tasks through a simple menu-driven interface.

## Features

- Add new tasks to the list
- View all saved tasks
- Remove completed or unwanted tasks
- User-friendly menu interface
- Input validation for task removal
- Continuous operation until the user chooses to exit
